# Energy-Aware A*

This repository contains the simulation code and dataset for our enhanced A* path-planning algorithm. Traditional A* focuses primarily on finding the shortest distance. This project introduces novel **energy-aware heuristics** designed to optimize battery consumption for autonomous robots, factoring in terrain difficulty, elevation, and turning costs, without significantly sacrificing path efficiency.
